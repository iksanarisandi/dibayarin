Kebutuhan API Key untuk ClawdBot

ClawdBot membutuhkan API key dari LLM provider untuk berfungsi. Pilihan utama:

1. OpenAI API (GPT-4 / GPT-4o)

  • Model: GPT-4, GPT-4 Turbo, atau GPT-4o (recommended)
  • Harga: Sekitar $0.01-0.03 per 1K tokens (input) dan $0.03-0.06 per 1K tokens (output)
  • Kelebihan: Model paling canggih, reasoning kuat, banyak fitur
  • Perkiraan biaya: $20-100/bulan untuk penggunaan personal moderate

2. Anthropic API (Claude 3.5 Sonnet / Opus)

  • Model: Claude 3.5 Sonnet (best value) atau Claude 3 Opus (premium)
  • Harga: Kompetitif dengan OpenAI, sekitar $3-15 per 1M tokens
  • Kelebihan: Context window besar (200K tokens), sangat baik untuk long-form tasks
  • Perkiraan biaya: $15-80/bulan untuk penggunaan moderate

3. Open-Source LLM (Local)

  • Model: Llama 3, Mistral, atau model lain yang bisa di-host sendiri
  • Harga: Gratis, tapi butuh spesifikasi VPS lebih tinggi (8GB+ RAM)
  • Kelebihan: Tidak ada biaya API, data sepenuhnya privat
  • Kekurangan: Kualitas masih di bawah GPT-4/Claude, butuh setup lebih kompleks

Cara Mendapatkan API Key

Untuk OpenAI API:

  1. Buat akun di platform.openai.com
  2. Go to Settings ‚Üí API Keys
  3. Create new secret key
  4. Set up payment method (kartu kredit internasional diperlukan)
  5. Set usage limit atau hard limit untuk kontrol biaya

Untuk Anthropic Claude API:

  1. Buat akun di console.anthropic.com
  2. Go to API Keys section
  3. Create new API key
  4. Set up payment method
  5. Set usage limits sesuai kebutuhan

Perkiraan Total Biaya Bulanan

  • API Usage: $20-100/bulan (tergantung seberapa sering ClawdBot digunakan)
  • Total: Sekitar $25-120/bulan atau Rp 400.000 - Rp 2.000.000
