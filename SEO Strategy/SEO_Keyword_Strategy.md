## ON-PAGE SEO CHECKLIST

Untuk setiap konten yang Anda buat, pastikan:

**Meta & Title**
- Meta Title: 50-60 characters, primary keyword di awal, action word (Best/How/Guide)
- Meta Description: 150-160 chars, primary + secondary keyword, CTA jelas

**Content Structure**
- H1: 1 per page, contain primary keyword, 50-60 chars
- H2/H3: Keyword natural, supporting long tails, semantic structure
- Content Length: Min 1,500+ words untuk competitive keywords, 800+ untuk long tail
- LSI Keywords: 10-15 related keywords naturally integrated

**Technical**
- URL Structure: Keyword included, lowercase, hyphenated, max 3 levels deep
- Image Optimization: Descriptive alt text + keyword, compressed, responsive
- Internal Linking: Min 10-15 links per 2,000 words, contextual + variations anchor text
- Schema Markup: Article schema, FAQPage, Organization, Product schema

**User Experience**
- CTA Elements: Clear call-to-action, prominent button, contact form above fold
- Mobile Optimization: AMP ready, touch-friendly, fast loading
- Core Web Vitals: LCP < 2.5s, FID < 100ms, CLS < 0.1
- Featured Snippet: Optimized answer (40-60 words), table/list format
